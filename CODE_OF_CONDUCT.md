# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful, assume good faith, and help keep this a welcoming
community for everyone. Harassment and discrimination are not tolerated.

Report concerns to the maintainers at the contact listed in the repository's
GitHub profile. Reports are handled confidentially.
